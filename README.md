# Benthic Rebloomed
A mod that aims to rework benthic bloom into a more consistantly good item.
Rather than "upgrading" item stacks to a higher rarity on stage advance, which can destroy a build, Benthic now leans into your build and upgrades stacks by adding more items to them:

### Benthic Bloom
Adds 2 items to 3 (+3 per stack) item stacks at the start of each stage. Corrupts all 57 Leaf Clovers.


## Config
The following can be configured:
* Number of stacks to upgrade (default 3)
* Number of items to add to each stack (default 2)
* If bloom can grant lunar items (default false)
* If bloom can grant more of itself (default false)
